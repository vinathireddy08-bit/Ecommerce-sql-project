 🛒 E-Commerce SQL Project

 📌 Project Overview
This project demonstrates database design and SQL analysis for an E-commerce platform.  
It includes schema creation, data insertion, and business analysis queries.

 🧰 Technologies Used
- MySQL
- SQL
- Git & GitHub
